# openCV-dLib-Cxx

My C++ implementation to problems on the online learnOpenCV blog.

Developed on MacOS 10.10 using g++ 5.4, openCV-3.4 and dLib-19.10

### Build and Run:

```
$ cd <folder>
$ cmake .
$ make 
$ ./<executable>
```

or

```
$ g++ -std=c++11 <file>.cpp `pkg-config --libs --cflags opencv` -o <executable>
$ ./<executable>
```
